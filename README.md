# Syncra Arg

Aplicación Flutter para gestión de liquidaciones de sueldos.

## Características
- Splash screen con logo y animación
- Navegación a selección de planes
- Gestión de usuarios y autenticación
- Cálculo de liquidaciones de sueldos
- Integración con Supabase

## Deploy en Vercel
Este proyecto está configurado para deploy automático en Vercel mediante integración con GitHub.

## Desarrollo
```bash
flutter pub get
flutter run -d web-server
```

## Build para producción
```bash
flutter build web
```
