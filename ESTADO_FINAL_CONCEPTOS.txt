1. CONCEPTOS REMUNERATIVOS
================================================================================

✅ SUELDO BÁSICO
   - Estado: IMPLEMENTADO
   - Cálculo: Proporcional según días trabajados (por defecto 30 días)
   - Código AFIP: 011000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Base Imponible: SÍ (topeado a $2.500.000)
   - Notas: Se deduce proporcionalmente si hay vacaciones activas

✅ PRESENTISMO / ASISTENCIA
   - Estado: IMPLEMENTADO
   - Cálculo: Porcentaje configurable (por defecto 8.33%)
   - Base: Sueldo básico proporcional
   - Código AFIP: 011000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Condición: Solo se paga si está activo y no hay inasistencias injustificadas

✅ HORAS EXTRAS 50%
   - Estado: IMPLEMENTADO
   - Cálculo: Automático según cantidad de horas ingresadas
   - Fórmula: (Sueldo Básico / Divisor Horas Mensuales) × 1.5 × Cantidad Horas
   - Divisor por defecto: 173 horas (Petroleros Jerárquicos)
   - Código AFIP: 051000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Notas: Cambiado de NO REMUNERATIVO a REMUNERATIVO (Enero 2026)

✅ HORAS EXTRAS 100%
   - Estado: IMPLEMENTADO
   - Cálculo: Automático según cantidad de horas ingresadas
   - Fórmula: (Sueldo Básico / Divisor Horas Mensuales) × 2.0 × Cantidad Horas
   - Divisor por defecto: 173 horas (Petroleros Jerárquicos)
   - Código AFIP: 051000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Notas: Cambiado de NO REMUNERATIVO a REMUNERATIVO (Enero 2026)

✅ ANTIGÜEDAD
   - Estado: IMPLEMENTADO (a través de servicios)
   - Cálculo: Según años de antigüedad y porcentaje del convenio
   - Código AFIP: 012000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Notas: Se calcula automáticamente según fecha de ingreso

✅ TÍTULO / PROFESIONALISMO

✅ VACACIONES
   - Estado: IMPLEMENTADO
   - Cálculo: Divisor 25 (Art. 155 LCT)
   - Fórmula: (Sueldo Básico / 25) × Días de Vacaciones
   - Código AFIP: 015000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ

     * Días de vacaciones configurables
     * Tipo: Gozadas / No Gozadas
     * Fechas de período vacacional (inicio y fin)
     * Deducción automática del sueldo básico para evitar doble pago
     * Ajuste manual disponible

✅ PLUS VACACIONAL
   - Estado: IMPLEMENTADO
   - Cálculo: Diferencia entre divisor 25 y 30
   - Fórmula: (Sueldo Básico / 25) - (Sueldo Básico / 30)
   - Código AFIP: 015000
   - Aporta a Jubilación: SÍ
   - Aporta a Obra Social: SÍ
   - Notas: Se calcula automáticamente cuando hay vacaciones activas

✅ SAC (AGUINALDO)

✅ KILÓMETROS RECORRIDOS (CCT Camioneros)

3. DEDUCCIONES / DESCUENTOS
================================================================================

✅ JUBILACIÓN (SIPA)
   - Estado: IMPLEMENTADO
   - Porcentaje: 11% sobre base topeada
   - Base de Cálculo: Mínimo entre Sueldo Bruto y TOPE_BASE ($2.500.000)
   - Código AFIP: 810000
   - Cálculo Automático: SÍ
   - Notas: Se aplica tope previsional obligatorio

✅ LEY 19.032 (PAMI)
   - Estado: IMPLEMENTADO
   - Porcentaje: 3% sobre base topeada
   - Base de Cálculo: Mínimo entre Sueldo Bruto y TOPE_BASE ($2.500.000)
   - Código AFIP: 810000
   - Cálculo Automático: SÍ
   - Notas: Se aplica tope previsional obligatorio

✅ OBRA SOCIAL
   - Estado: IMPLEMENTADO
   - Porcentaje: 3% sobre base topeada
   - Base de Cálculo: Mínimo entre Sueldo Bruto y TOPE_BASE ($2.500.000)
   - Código AFIP: 810000
   - Cálculo Automático: SÍ
   - Notas: Se aplica tope previsional obligatorio

✅ CUOTA SINDICAL
   - Estado: IMPLEMENTADO
   - Porcentaje: 2.5% sobre base topeada
   - Base de Cálculo: Mínimo entre Sueldo Bruto y TOPE_BASE ($2.500.000)
   - Código AFIP: 820000
   - Cálculo Automático: SÍ
   - Condición: Solo si el empleado está afiliado sindicalmente
   - Notas: Se aplica tope previsional obligatorio

✅ IMPUESTO A LAS GANANCIAS (4ta Categoría)
   - Estado: IMPLEMENTADO
   - Cálculo: Automático con validación obligatoria
   - Base de Cálculo: Remuneración Neta Sujeta a Impuestos (base topeada)
   - Mínimo No Imponible: 15 × SMVM = $5.115.000 (Enero 2026)
   - Escala Progresiva:
     * Hasta $2.000.000 de excedente: 27%
     * Más de $2.000.000 a $5.000.000: 30%
     * Más de $5.000.000: 35%
   - Código AFIP: 990000
   - Modo Manual: Disponible (puede desactivarse cálculo automático)
   - Validación: Si Remuneración Neta < Mínimo No Imponible, retención = 0
   - Notas: Sistema completo con cálculo dinámico según legislación vigente

✅ FONDO DE DESEMPLEO (UOCRA)

7. CARACTERÍSTICAS ESPECIALES
================================================================================

✅ CÁLCULO PROPORCIONAL
   - Días trabajados configurables (por defecto 30 días)
   - Aplicación proporcional a sueldo básico y presentismo
   - Compatible con liquidaciones parciales

✅ TOPE PREVISIONAL
   - Valor: $2.500.000 (Enero 2026)
   - Aplicación: Obligatoria para todos los aportes
   - Base de Cálculo: Mínimo entre Sueldo Bruto y TOPE_BASE
   - Afecta: Jubilación, PAMI, Obra Social, Cuota Sindical

✅ DEDUCCIÓN DE VACACIONES DEL SUELDO BÁSICO
   - Implementado según Art. 155 LCT
   - Evita pagar los mismos días dos veces
   - Se muestra explícitamente en la tabla de liquidación

✅ CÁLCULO AUTOMÁTICO DE GANANCIAS
   - Sistema completo con validación obligatoria
   - Escala progresiva según legislación vigente
   - Modo manual disponible
   - Validación de mínimo no imponible

✅ GESTIÓN DE INASISTENCIAS
   - Días de inasistencia injustificada configurables
   - Afecta el cálculo de presentismo
   - Presentismo solo se paga si no hay inasistencias

✅ SISTEMA DE VACACIONES COMPLETO
   - Días configurables
   - Tipo: Gozadas / No Gozadas
   - Fechas de período vacacional
   - Cálculo con divisor 25
   - Plus vacacional automático
   - Deducción del sueldo básico

✅ EXPORTACIÓN ARCA LSD
   - Compatible con formato ARCA LSD
   - Generación de archivos de importación de conceptos
   - Mapeo automático de códigos AFIP
   - Validación de estructura de archivos
